# Merchant icons

Bundled for offline merchant recognition in the MOSS Money ledger. No runtime network calls.

## Simple Icons (MIT)

Most brand SVG paths are from [Simple Icons](https://simpleicons.org/) (MIT License).
See https://github.com/simple-icons/simple-icons/blob/develop/LICENSE.md

## Handcrafted fallbacks

When Simple Icons has no entry, MOSS ships a minimal brand-colored badge (initials on a solid
swatch) for common US merchants — e.g. Costco, CVS, Whole Foods. These are original MOSS assets,
not third-party trademark artwork.
